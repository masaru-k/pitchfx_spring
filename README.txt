"pitchrx data prep upload version" uses pitchrx and mlbgameday packages to scrape games between 2013 and 2018, then calculates game-by-game pitcher stats from those games.
The code requires the pitchrx and mlbgameday packages to be installed and the 'woba_event_list.csv' file in the working directory
"spring vel drop model code" produces the stats and analysis used in the articles
